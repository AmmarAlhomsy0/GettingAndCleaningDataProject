# Getting and Cleaning Data - Course Project

## Overview
This project processes the **UCI HAR Dataset** to create a tidy dataset.

## Files Included
- `run_analysis.R`: The script that processes the dataset.
- `tidy_dataset.txt`: The final cleaned dataset.
- `CodeBook.md`: A description of variables and transformations.
- `README.md`: This file, explaining the project.

## How to Run
1. Download and extract the dataset.
2. Run the script:
   ```r
   source("run_analysis.R")
